### ✨ Key Features
* 📊 **Clean Dashboard** — Monitor stok & antrean dalam satu layar.
* 📖 **Smart Catalog** — Manajemen buku fisik yang terorganisir.
* 📝 **Transaction Log** — Alur peminjaman & pengembalian simpel.
* 🛠️ **Staff Control** — Manajemen admin yang aman.

### 🚀 Quick Start
1. **Clone** project ke htdocs.
2. **Import** database `bibliotech.sql`.
3. **Configure** `config/Database.php`.
4. **Login** & kelola perpustakaanmu!
